Pennies converter

This is a sample page on how the library works.

This library contains two basic modules:

  • A validator, this is used to validate the input to be correct.
  • A converter, this is used to calculate the right value to be returned using an appropriate format.

The whole thing is handled by the main script that acts as a controller.

See the Jasmine tests to see everything is working as expected on a lower level.
